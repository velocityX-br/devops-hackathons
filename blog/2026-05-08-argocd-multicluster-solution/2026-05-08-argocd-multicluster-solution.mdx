ArgoCD `ApplicationSet`s `Matric Generator`. Cartesian product (笛卡尔积).

Application List x Cluster List = Autogenerated multiple Applications

Dynamically apply to the clusters based on the selector conditions.

```
apiVersion: argoproj.io/v1alpha1
kind: ApplicationSet
metadata:
  name: appset-core-services

# reference
# https://argoproj.github.io/argo-cd/operator-manual/cluster-bootstrapping/#applicationset
# https://argoproj.github.io/argo-cd/operator-manual/cluster-bootstrapping/#applicationset-templating
# https://argoproj.github.io/argo-cd/operator-manual/cluster-bootstrapping/#applicationset-templating-variables
# https://argoproj.github.io/argo-cd/operator-manual/cluster-bootstrapping/#applicationset-templating-functions

spec:
  generators:
    - matrix:
        generators:
          - list:
              elements:
              - service_name: redis
                path: argocd/helm_charts/redis
                namespace: redis
                helm_dependency_chart_name: redis
              - service_name: grafana
                path: argocd/helm_charts/monitoring/grafana
                namespace: grafana
          - clusters:
              selector:
                matchExpressions:
                  - {key: usecase, operator: In, values: [gitops]}
                  - {key: istio-ready, operator: In, values: ["true"]}
    - matrix:
        generators:
          - list:
              elements:
              - service_name: gitlab
                path: argocd/helm_charts/gitlab
                namespace: gitlab
                helm_dependency_chart_name: gitlab
              - service_name: patching
                namespace: patching
                path: argocd/helm_charts/patching
          - clusters:
              selector:
                matchExpressions:
                  - {key: usecase, operator: In, values: [gitops]}
                  - {key: istio-ready, operator: In, values: ["true"]}
                  - {key: stags, operator: In, values: ["final"]}
```
